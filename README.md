# 예담 최종프로젝트, '스크린야구 예약 시스템'
### 예담직업전문학교에서 팀 단위로 실시한 최종프로젝트 결과물입니다.
### 저희팀은 '스크린야구 예약 시스템' 이라는 주제로 프로젝트를 만들게 되었습니다.
### 그 중 제가 맡은 역할은, 관리자 페이지에서 전반적인 매장관리 파트를 맡게되었습니다.

```sh
* 근무자 관리
* 매출관리
* TodoList 관리
```

> Note: `--capt-add=SYS-ADMIN` is required for PDF rendering.
